# 物流轨迹中心 · prototype-draft

> 原页面参考：`https://erp.cloudebox.cn/yh-logistics/logisticsTrajectory`

## 页面目标

在一个包裹监控页面中统一查询正向物流和逆向退件，按物流方向切换对应状态，点击包裹打开详情抽屉查看完整生命周期。

## 本轮页面清单

| 页面 | 主任务 | 本轮范围 |
|---|---|---|
| 物流轨迹中心 | 发现物流异常并查看包裹轨迹 | 正向/逆向切换、动态状态、筛选、列表、详情抽屉 |

## 明确不做

- 不新增退件仓入库、质检、退款操作页。
- 不生成 Figma 文件，不发布在线链接。
- 不接入真实接口；列表和轨迹为可交互 Mock 数据。
- 不在正式页面中增加角色切换、状态切换或调试面板。

## 页面来源映射

| 可见元素 | 来源 | 说明 |
|---|---|---|
| 物流方向 Segmented | 本轮调整 | 正向 / 逆向；不展示全部包裹和异常待处理 |
| 关键指标 | 本轮已确认方案 | 根据方向动态切换指标 |
| 筛选区 | 原页面线上实现 | 仓库、平台、目的国家、物流商、物流渠道、平台单号、交运时间；保留查询与重置 |
| 状态 Tab | 原页面线上实现 + 本轮改造 | 全部、未上网、已上网、已交航、已到达目的国、已签收、包裹异常、退回；派送失败归入包裹异常类型 |
| 异常类型二级筛选 | 本轮改造 | 正向包裹异常下区分：查不到轨迹、派送失败、丢件、破损、轨迹同步失败；派送失败说明重新派送后回到已签收 |
| 退回节点二级筛选 | 本轮改造 | 正向退回下新增：退回中、退回签收、退回破损丢件；正向只跟踪原包裹物流轨迹 |
| 包裹列表 | 原页面线上实现 + 改造 | 保留原始列字段；逆向模式切换为退件字段；增加固定操作列 |
| 批量操作 | 原页面线上实现 | 批量修改备注、导入、数据导出（选中数据 / 当前条件） |
| 轨迹详情抽屉 | 原页面 `trajectory` 弹窗 + 改造方案 | 保留物流名称、物流单号、轨迹时间线；补充正向 / 逆向 / 包裹信息 |
| 轨迹同步失败 | 截图现有技术异常 | 列表业务化显示，技术堆栈不直接暴露 |

## UI 设计契约

- 默认使用中文 B 端 ERP 密度：浅灰布局背景、白色面板、细边框、紧凑表格。
- 基础颜色、字体、间距、圆角和阴影全部引用 `../../ui-library/tokens.css`。
- 物流方向使用 Segmented 语义；状态使用 Tabs 语义；标签必须同时使用 `.tag` 与 `.tag--*`。
- 本轮调整移除页面级统计卡片；正向列表移除“方向 / 状态”独立列，状态统一在“最新轨迹信息”中表达。
- 正向列表新增“揽收时间”，位置在“最新轨迹信息”之后、“交运时间”之前。
- 逆向列表移除“退件申请时间、买家寄出时间、退件入库时间、质检时间、退款时间、退件时效”，改为“退回时间、签收时间”。
- 逆向列表移除“售后单号、退件包裹号”，改为“平台单号、物流单号”。
- 列表不展示承运商技术堆栈；技术错误统一映射为“轨迹同步失败”，详情中显示可理解的处理建议。
- 查询优先使用列表和详情抽屉，不跳转新页面。
- 详情抽屉默认从右侧打开，展示包裹基础信息、正向/逆向轨迹和处理提示。

## 交互与状态覆盖

| 状态 | 原型表现 |
|---|---|
| 默认 | 正向物流、正向状态 Tab、包裹列表 |
| 正向物流 | 状态 Tab 和列表切换到正向语义 |
| 逆向物流 | 状态 Tab 和列表切换到退件语义，仅保留退回中、已签收、包裹异常节点；包裹异常类型为妥投失败、破损丢件；下方仅显示退回类型筛选 |
| 包裹异常类型 | 点击“包裹异常”后展示异常类型二级筛选，支持按类型过滤列表；正向覆盖丢件、轨迹同步失败等类型 |
| 退回节点 | 正向点击“退回”后展示退回节点筛选；逆向视图展示退回类型筛选 |
| 筛选 | 关键词支持订单号、包裹号、运单号匹配；查询按钮刷新列表 |
| 详情 | 点击包裹号或“查看轨迹”打开抽屉；支持 ESC 和遮罩关闭 |
| 无结果 | 表格展示筛选空状态 |
| 同步/导出/重试 | 使用 Toast 反馈原型动作结果 |

## Mock 数据说明

- 正向样例覆盖：待上网、运输中、清关异常、已签收、轨迹同步失败、查不到轨迹、派送失败、丢件、破损、退回中、退回签收、退回破损丢件。
- 逆向样例覆盖：退回中、已签收、包裹异常；包裹异常类型覆盖妥投失败、破损丢件。
- 页面展示的总数沿用截图中的量级，具体行数据为演示数据。

## 原页面字段保留

正向列表保留以下原始字段：仓库、平台、物流商、渠道、国家、平台单号、包裹号、物流单号、物流跟踪号、最新轨迹信息、交运时间、上网时间、交航时间、到达目的国时间、签收时间、签收时效、备注。

原页面点击包裹号后打开“物流轨迹”弹窗，展示物流名称、物流单号和 `trackingList` 时间线；本原型保留该入口，并在详情中增加正向 / 逆向轨迹切换。
